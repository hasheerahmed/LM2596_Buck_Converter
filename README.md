# LM2596 Buck Converter PCB

## Project Objective

Design a manufacturable 2-layer PCB for a buck converter using the LM2596S-5.0 switching regulator.

### Requirements

Input Voltage:
7V - 24V DC

Output Voltage:
5V DC

Output Current:
1A minimum

PCB:
2-layer

Package:
TO-263-5

Target Manufacturer:
JLCPCB

---

## Design Decision 1

Selected IC:
LM2596S-5.0

Reason:

* Widely used buck regulator
* Extensive documentation available
* Suitable for learning power electronics PCB design
* Available in surface mount TO-263 package
